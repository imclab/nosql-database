iBoxDB

fast embeddable transactional table-style document nosql database

Project Description

iBoxDB is a fast embeddable transactional table-style document nosql database, easily process objects and documents,traditional table with unstructured data, zero configuration, pure java/.net database, no dependencies.

Features

CURD, index, composite index, sql-like query language, transactions, mvcc, concurrency control, master-master-slave-replication, in-memory database, dynamic columns, nosql ORM, high performance.

Supports

java, .net, android, windows phone, mono, compatible with unity3d, node.js.
