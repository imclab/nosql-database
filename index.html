iBoxDB is a fast embeddable transactional table-style document nosql database, easily process objects and documents, zero configuration, pure java/.net database, no dependencies

Features: CURD, index, composite index, query language, transactions, mvcc, concurrency control, master-master-slave-replication, in-memory database, dynamic columns, nosql styles, high performance, traditional table with unstructured data.

Supports java, .net, android, windows phone, mono, compatible with unity3d, node.js
